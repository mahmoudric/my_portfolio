## 🚀 Want to Run It Locally?

### What You Need
- Node.js (version 18 or newer)
- pnpm (a fast package manager)

### Let's Get Started

```bash
# Clone the repo (if you have it)
git clone <repo-url>
cd designer_portfolio

# Install dependencies
pnpm install

# Start the development server
pnpm dev

# Open your browser to http://localhost:3000
```

The development server automatically reloads when you make changes—no need to refresh!

### Building for Production

```bash
# Create an optimized build
pnpm build

# Test the production build locally
pnpm start
```

## 🐛 Something Not Working?

### Buttons acting weird?
- Check the browser console (F12) for error messages
- Try clearing your browser cache

### Images not showing?
- Make sure they're uploaded to cloud storage
- Check the image URLs in the code

### Build failing?
- Run `pnpm install` again to make sure all dependencies are there
- Make sure you have Node.js 18+
